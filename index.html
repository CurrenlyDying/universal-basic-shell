UBS — Universal Basic Shell

GitHub Pages control center for routing, generator, and PWA workflows.

Architecture

  • Tier 0: Apps Script proxy via Google infrastructure.
  • Tier 1: Cloud Run signaling + Colab/Cloud Shell worker browser node.
  • Tier 2: Headscale/WireGuard mesh.
  • Tier 3: Volunteer residential relay nodes.

Use this portal to navigate every core UBS flow from one place on GitHub Pages.

Step-by-step user flow

  1. Open the PWA Generator tab and analyze your target service.
  2. Generate recipe + manifest + service worker artifacts.
  3. Open the Universal PWA Shell tab to route browsing traffic through UBS.
  4. Use Operator Tools to verify signaling server and worker node availability.
